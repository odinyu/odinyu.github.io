最近几天折腾的几个网站和工具

Odin Yu
2024-08-09
折腾 › handyInfo
clourflare, docker

我的 99 元主机有一段时间没有去管它了，主要是因为毕竟它因为便宜所以性能有限，可玩性不强。不过本着既然已经花钱了，也不能让它浪费的原则，这几天又开始在上面简单地折腾一下。

miniserve

在 github 的一个自托管应用推荐的 repo 中挑到了一个简单地临时目录或文件的分享工具 - miniserve，它的主要作用就是如果临时想分享个服务器上的某个路径或文件，就可以利用它来轻松做到，并且可以随时关闭。官方文档中对于用 docker 方式部署和使用介绍得有点简单，自己折腾了一下，还算满意，如下截图：

从 QuantumultX 开始的折腾

从上面那个博客里看到 QuantumultX，不明所以。问了一下 GPT，告诉我。

Quantumult X 是一款适用于 iOS 和 macOS 系统的代理软件客户端，支持多种代理协议如 Shadowsocks、ShadowsocksR、HTTP、Socks5 和 VMess。它功能强大，用于网络调试、去广告和抓包等操作。Quantumult X 是一款付费应用，售价为 7.99 美元。

然后再详细地看了下它的官网，要在美区商店里才能下载，所以就只能作罢了。不过类似 QuantumultX 的服务其实对我来说还是有点用的，所以有没有白嫖的那种呢？

结合自己已经白嫖的图床，在 github 上搜索到一堆可创建在 Cloudflare 上的 proxy，花了10分钟搞定。试用了一下，然后又换了几个类似的，最终可以无痛地访问 reddit、github、v2ex等网站，用来防身还是蛮不错的。
