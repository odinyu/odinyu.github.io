像屎一样的A股

Odin Yu
2022-09-20

没错，A股它就是砣屎！

2022年的A股，从1月开始就一无反顾地向下滑，一直滑到4月26日。然后从7月的下半月再次往下滑，让本来以为经过5、6月份的反弹有了点暖意的A股，再次遍体寒意。

股票投资，2020年~2021年的时候觉得还是有利可图的，但就2022年目前的情况来看，是真的难做！白马股会崩、抱团股会崩、热门股一样也会崩。所以，不管是价值投资还是价值投机或者是直接投机，今年的行情都是一个字，难！

做点正经事

目前在学SwiftUI，一种描述式的构建 UI 的⽅式（私以为这种描述挺到位的）。一直有个想做APP的想法，可惜对于 UI 始终没什么感觉，也就是没什么美感。尝试过 Android 、 electron ，但是都无疾而终。

说回 SwiftUI 吧，既然它是构建 UI 的方式，并且又是 Apple 官方支持的编程语言，所以还是挺有学习价值的。目前是在跟着官方的 tutorial 在做练习，不得不说官方的教程做得太好了！那种动效简直了！

另外前几天也在学 Vue ，这两天有点放松所以先放放，不过还是要继续学的。嗯，时间不大够啊，又要工作、又要娱乐、还要家庭生活，所以分给学习的时间是有点少。

近期的项目

  • 上汽财务二期，这期的开发量还是有一点的，不太确定那小子能不能搞定，所以我可能会帮他一起做掉一些基础的开发工作。
  • 六院，暂时没什么开发，但后续不太确定
  • 宁波文旅，准备交给另一个小子，虽然觉得他应该是可以胜任，不过是他的第一个项目还是盯得紧一点
  • 迪卡侬，已知快递查询这部分还要修改，工作量应该不大
  • 上海人社，知识点同步（最麻烦的功能），前提条件目前还不大够，看项目经理怎么去 PK 了

其他

儿子的高中生活已经正式展开，暂时看他还是蛮认真的，今天晚上还去参加生物竞赛的测试。希望新的环境会带给他不同的感觉，毕竟人生的起点就在此刻吧。

老婆天天和我抱怨她自己蛮胖了，我觉得她其实还好，肉乎乎的蛮舒服的呀，哈哈！

下班下班！
